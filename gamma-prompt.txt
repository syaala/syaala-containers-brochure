DESIGN EXECUTION NOTES FOR GAMMA
───────────────────────────────────────────────────────────────────

1. IMAGE TREATMENT:
   - All AI-generated images should use organic curved masks
   - Blend images into gradients, never hard edges
   - Use jagged/rough image edges intentionally as design element
   - Apply subtle blur gradients at image boundaries
   - Layer images with transparency for depth

2. COLOR TEMPERATURE:
   - Compute/Power slides: Warm (orange #FF9D5C, amber)
   - Cooling slides: Cool (light blue #5CC8FF, cyan)
   - CTA/Highlights: Neon green #00FFA8
   - Base: Deep blacks with warm gray text

3. TYPOGRAPHY:
   - Headlines: 60-120pt, bold, dramatic
   - Subheads: 24-36pt, medium weight
   - Body: 16-20pt, regular, high line-spacing
   - Never use rigid alignment - embrace organic flow

4. MOTION (if animated):
   - Smooth parallax scrolling
   - Fade-in with subtle upward drift
   - Glow pulse effects on accent elements
   - Avoid sharp cuts - use gradient transitions

5. PREMIUM TOUCHES:
   - Glass morphism cards (frosted background, subtle blur)
   - Soft drop shadows for depth (not harsh)
   - Gradient overlays on images for text legibility
   - Subtle grain texture for cinematic feel
   - Generous white space - premium spacing
